# 도형 드로잉 프로그램
기존 라이브러리에서 제공하는 도형들이 어떻게 그려지는지, 직접 픽셀 단위로 구현했습니다.

### Algorithms
- 직선 알고리즘
  - Bresenham's line algorithm
  - Digital Differential Analyzer(DDA)
- 원 알고리즘
  - Bresenham's Circle Algorithm
- 도형 위치 이동, 크기 변환, 회전
  - 선형대수학 행렬 연산을 이용하여 구현

### Components
- Java(Swing)
